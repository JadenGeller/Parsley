  • many(_:then:)

    Constructs a Parser that will run parser 0 or more times, as many times as possible, and then will run then. If then fails, the parser will undo runs of parser until then succeeds, otherwise will throw a ParseError. On success, returns a tuple with the resulting array from applying parser on the left and with the result of then on the right.

    Parameter

    Parameter parser: The parser to run repeatedly.

    Parameter

    Parameter then: The parser whose success is desired after applications of parser.

    Throws

    ParseError if then does not succeed for any number of times that parser succeeds.

    Declaration

    Swift

    The parser whose success is desired after applications of parser.
